# StudySmart AI 🎯

An intelligent study tracking system with AI-powered analytics to optimize your learning habits.

**Built with:** Python, Flask, HTML/CSS/JS, Pandas, NumPy

## Features
- 📚 Track study sessions & distractions
- 📊 Real-time productivity dashboard  
- 🤖 AI-powered trend analysis
- 🔮 Weekly performance predictions
- 💡 Personalized study recommendations
